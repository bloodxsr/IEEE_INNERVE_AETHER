# Artefakt.mov — Full Website Build Prompt

## Project Overview
Build a dark, cinematic, single-page website for a hybrid film production studio called **artefakt.mov** 
based in Frankfurt, Germany. The aesthetic is minimal, editorial, and brutalist-luxury — black 
background, white text, monospaced fonts, heavy whitespace, and smooth intentional transitions. 
Every section should feel like a high-end film production zine.

---

## 🎨 Design System

### Colors
- Background: `#000000` (pure black)
- Primary text: `#f0f0f0` (off-white)
- Secondary text: `#888888` (muted gray)
- Borders/rules: `#1a1a1a` or `#222222`
- Accent: none — no color, black and white only

### Typography
- **Display/Headings:** Neue Haas Grotesk or Inter — bold, tight letter-spacing, large scale
- **Body:** Inter Regular — comfortable line-height (1.6–1.8)
- **Labels/Codes:** Space Mono or Courier New — monospaced, small size, uppercase
- Load from Google Fonts: `Inter` (weights 300, 400, 700) + `Space Mono` (400)

### Spacing & Layout
- Max content width: `1400px`, centered
- Section padding: `160px 80px` desktop, `80px 24px` mobile
- Thin horizontal `<hr>` rules (`1px solid #1a1a1a`) between sections
- Heavy use of vertical whitespace — breathe, don't crowd

### CSS Custom Properties (define at `:root`)
```css
:root {
  --bg: #000000;
  --text: #f0f0f0;
  --muted: #888888;
  --border: #1a1a1a;
  --font-display: 'Inter', sans-serif;
  --font-mono: 'Space Mono', monospace;
  --transition-fast: 300ms ease;
  --transition-med: 500ms cubic-bezier(0.25, 0.1, 0.25, 1);
  --transition-slow: 900ms cubic-bezier(0.16, 1, 0.3, 1);
}
```

## 🧭 Navigation

### Structure
- Fixed top bar, full width, `z-index: 100`
- Background: transparent initially, stays transparent (no blur/fill on scroll)
- **Left:** Logo mark `:/` in Space Mono font, white, ~18px
- **Right:** Text button that reads `Menu` in small caps

### Menu Overlay — Full Screen
- On click `Menu`: a full-screen black overlay **slides in from the right** (translateX: 100% → 0%)
- Transition: `400ms cubic-bezier(0.76, 0, 0.24, 1)`
- Overlay content (large centered links, vertical stack):
  - Works
  - About
  - ——————
  - Instagram ↗
  - info@artefakt.mov
- Each nav link **staggers in** after overlay arrives: opacity 0→1 + translateY 20px→0, 80ms apart
- Top right: `Close` button replaces `Menu`
- On close: overlay slides back out to the right, links fade out first

## ✨ Animation & Transition Master Reference

| Element | Animation | Timing |
|---|---|---|
| Page load — hero headline | translateY(60px)→0, opacity 0→1 | 900ms, staggered 120ms per line |
| Page load — hero paragraph | opacity 0→1 | 900ms, 400ms delay |
| Menu overlay open | translateX(100%)→0 | 400ms cubic-bezier(0.76, 0, 0.24, 1) |
| Menu overlay close | translateX(0)→100% | 400ms reverse |
| Menu nav links stagger | translateY(20px)→0, opacity 0→1 | 80ms per link, starts after overlay |
| Section labels on scroll | opacity 0→1 | 400ms ease |
| Works carousel slide | translateX per slide | 500ms cubic-bezier(0.25, 0.1, 0.25, 1) |
| Identity words stagger | translateY(40px)→0, opacity 0→1 | 80ms per word |
| Expertise columns stagger | opacity 0→1 | 150ms stagger per column |
| All link hovers | underline from left | 300ms ease |
| Parallax image | translateY at 0.2x scroll | requestAnimationFrame |
| Back to top scroll | smooth native scroll | browser native |

All scroll-triggered animations use **IntersectionObserver** with `threshold: 0.15`.

## 🛠️ Technical Requirements

- **No heavy JS libraries** — vanilla JS only for interactions
- **IntersectionObserver** for all scroll-triggered animations
- **CSS transitions/keyframes** only — no GSAP unless Orchids supports it natively
- Carousel: touch events (`touchstart`, `touchmove`, `touchend`) for swipe
- Images: use dark placeholder images (`#111111` background with subtle grain texture)
- Videos: muted, autoplay, loop, playsinline
- Smooth scroll: `html { scroll-behavior: smooth; }`
- Fully semantic HTML: `<nav>`, `<main>`, `<section>`, `<footer>`
- No horizontal scroll at any viewport width
